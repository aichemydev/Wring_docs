Capture API Calls

The Wring API call interceptor can be used to intercept API calls made during the operation of your web app. After these calls have been captured, you can use the Wring apps dashboard to analyze them.

Installing the JS snippetâ€‹

Include the following snippet in the <head> element of your index.html or main template HTML file:

<script src="https://wringapiutils.s3.us-west-2.amazonaws.com/interceptor-packages/interceptor.min.js"
  integrity="sha384-"
  crossorigin="anonymous"></script>
<script>
  wringInterceptor.init("phc_XXXX");
</script>

Replace the phc_XXXX item above with the API key from a Wring apps project that is already being used to capture user actions on your web app, for example:

In this way, you will be able to analyze API calls and user actions at the same time.
