Instructions:

This is how it looks in browser for Calendar:

  1. Go to Expander/Setup and you will see all the steps

  1. . Edit all select actions â€“ Explore from No to Yes ( Explore: Yes)

  1. After the second select action, add manually Save and verify_contains steps from Add button.

  1. Add these steps manually:
  • SAVE "//*[@id="__next"]/div/div[1]/div/div/div[2]/div/div[2]/div/span" into "duration"
  • SAVE "//a[contains(string(), "Confirm")]/preceding-sibling::a/div[1]" into "timeslot"
  • VERIFY_CONTAINS css selector "div.af-confirm" - ${timeslot} + ${duration}
